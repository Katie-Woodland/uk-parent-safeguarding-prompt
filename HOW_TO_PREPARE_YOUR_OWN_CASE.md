# How to Prepare Your Own Case for the Prompt

### Recommended File Naming & Structure
1. `CHRONOLOGY.md` or `My_Timeline.docx` — your own dated list of events
2. All emails → save as PDF or text with clear dates
3. School reports / CAMHS reports / EHCP / assessments
4. Attendance records / exclusion letters
5. Any policies the school sent you

### Best Practice
- Redact all names, addresses, exact dates if you prefer extra privacy (the prompt can still work).
- Organise files clearly (use numbers: 01_Email_Headteacher.pdf, 02_CAMHS_Report.pdf).
- Write a short 1-paragraph summary of the main problem when you start the chat.

The clearer your evidence, the better the analysis.
