# Contributing to UK Parent Safeguarding Prompt Toolkit

Thank you for considering contributing to this project!  
This toolkit exists to help UK parents for **free, forever**. We welcome contributions that improve the tool while keeping it safe, accurate, and non-commercial.

## How Can You Contribute?

### 1. Reporting Issues
- Found a bug in the prompt?
- Have a suggestion for improvement?
- Noticed something unclear in the README or examples?

→ Open an **Issue** with a clear title and description.

### 2. Submitting Changes (Pull Requests)
We welcome:
- Improvements to the Master Prompt
- Additional template letters or complaint wording
- Updated policy references (England, Wales, Scotland, Northern Ireland)
- Anonymised real case examples (heavily redacted)
- Better formatting, tables, or Stage 6 summary layout
- Bug fixes or clarity enhancements

### 3. Adding New Content
**Before contributing any case examples or templates:**
- All personal data **must be fully anonymised** (no names, dates, locations, school names, etc. that could identify anyone).
- Remove or generalise any sensitive details.
- Add a note at the top: “Example case – fully anonymised”.

### Contribution Guidelines
- Keep the tone professional and neutral at all times.
- Maintain the “100% factual, no emotion” principle in the prompt.
- Test any prompt changes with at least one sample case.
- Update the version number in `PROMPT.md` if making significant changes (e.g. 1.2).
- All contributions must respect the [CC BY-NC-SA 4.0](LICENSE) license.

### Development Workflow
1. Fork the repository
2. Create a new branch (`git checkout -b improvement-xyz`)
3. Make your changes
4. Submit a Pull Request with a clear description
