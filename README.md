# UK Parent Safeguarding Prompt Toolkit

**Free AI-powered tool for UK parents navigating education and safeguarding systems.**

This repository contains a carefully engineered, iterative prompt that turns any large language model (Grok, Claude, GPT, etc.) into a **professional, evidence-based analyst** for school complaints, SEND issues, social services involvement, and CAMHS cases.

### Why This Exists
Parents across the UK repeatedly face procedural failures, misapplied labels, inadequate support, and closed ranks when trying to get help for their children. This prompt helps them:
- Build clear chronological timelines from their evidence
- Identify specific policy and statutory breaches
- Produce professional, non-emotional reports
- Understand correct escalation pathways
- Request label reviews or gaps assessments

Developed from real clinical psychology and advocacy cases by **Katie Woodland**.

### Features
- Stage-by-stage analysis (Timeline → Policies → Breaches → Recommendations)
- Covers England & Wales law and guidance (with notes on differences)
- Optional diagnostic/label review or support gaps assessment
- Professional Executive Summary suitable for governors, local authorities, Ombudsmen, or tribunals
- Fully customisable and iterative

### How to Use
1. Copy the full prompt from [`PROMPT.md`](PROMPT.md)
2. Start a new chat with any strong AI model
3. Upload your documents (emails, reports, policies, chronologies, attendance records, etc.)
4. Run stage by stage — the prompt will guide you

**Important**: This is a support tool only. Always review the output yourself and seek professional legal advice where needed. It is not a substitute for qualified legal representation.
